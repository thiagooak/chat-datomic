Tutorial
========

This tutorial demonstrates iteratively building an inventory database. The tutorial comprises seven steps:

- an overview of the [Client API](client.html)
- creating an initial data model with [assertions](assertion.html)
- querying a database value with [datalog and pull](read.html)
- adding [new schema and data](accumulate.html) to meet new requirements
- more powerful queries with [parameters and rules](read-revisited.html)
- modeling change over time with [retractions](retract.html)
- point in time queries and auditing with [as-of and history](history.html)
