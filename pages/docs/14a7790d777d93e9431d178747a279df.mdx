Client Library Setup
====================

You can connect to Datomic via [clients or peers](../overview/clients-and-peers.html). First time users should follow the [Getting Started](../getting-started/get-datomic.html) process using Peers.

When you're ready to integrate the [Datomic Client](https://docs.datomic.com/client-api/datomic.client.api.html) into your project, this document provides details on how to setup your Client library dependencies. For analagous information for using the Peer Library in your project see [Integrating the Peer Library](../peer/integrating-peer-lib.html).

Client Setup
------------

The Datomic Client library is provided via [ Maven Central.](https://search.maven.org/artifact/com.datomic/client-pro)





Project Configuration
---------------------



- <a id="maven"></a>Maven  
    To retrieve the Client library for a Maven project, add the following snippet inside the &lt;dependencies&gt; block of your pom.xml file:
    
    ```
    <pre class="src src-xml"><<span style="color: #0000ff;">dependency</span>>
      <<span style="color: #0000ff;">groupId</span>>com.datomic</<span style="color: #0000ff;">groupId</span>>
      <<span style="color: #0000ff;">artifactId</span>>client-pro</<span style="color: #0000ff;">artifactId</span>>
      <<span style="color: #0000ff;">version</span>>1.0.76</<span style="color: #0000ff;">version</span>>
    </<span style="color: #0000ff;">dependency</span>>
    ```
    
    
    
    Replacing the version number as appropriate.
- <a id="deps"></a>Clojure CLI  
    To use the Client library from a [Clojure CLI REPL](https://clojure.org/guides/deps_and_cli), add the following to your *deps.edn* dependencies map:
    
    ```
    <pre class="src src-clojure">com.datomic/client-pro {<span style="color: #008b8b;">:mvn/version</span> <span style="color: #8b2252;">"1.0.76"</span>}
    ```
    
    
    
    Replacing the version number as appropriate.
- <a id="leiningen"></a>Leiningen  
    To include the Client library in a Leiningen project, add the following snippet to your *project.clj* file in collection under `:dependencies` key
    
    ```
    <pre class="src src-clojure">[com.datomic/client-pro <span style="color: #8b2252;">"1.0.76"</span>]
    ```
    
    
    
    Replacing the version number as appropriate.
