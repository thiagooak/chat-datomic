Datomic Software
----------------

Datomic Cloud software includes:

- CloudFormation templates to [start](operation/new-system.html) or [upgrade](operation/upgrading.html) a system
- the [Client API library](client/client-api.html)
- [Ion libraries](ions/ions-reference.html)
- the [Datomic CLI tools](operation/cli-tools.html)

Track dev-local releases via the [dev-local documentation](dev-local.html).
